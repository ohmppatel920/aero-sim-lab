# Rocket demo recording script (60–90 seconds)

1. Open app: `streamlit run rocket/app.py`
2. Show baseline metrics: apogee, range, max-q, impact time.
3. Increase thrust — note apogee and range change; mention energy addition during burn.
4. Increase Cd — note performance loss and max-q behavior.
5. Adjust thrust align speed slider: high value (rail-like) vs low value (velocity-aligned) and compare trajectory shape.
6. Close with one limitation from `docs/model.md` (e.g., no 6-DOF attitude dynamics).

Save recording as `docs/demo-rocket.mp4` or link from portfolio README when ready.
